🚀 CODSOFT Machine Learning Internship Projects

This repository contains 3 end-to-end Machine Learning projects completed as part of my CodSoft Internship. Each project includes data preprocessing, model training, evaluation, and a user-friendly interface for predictions.

📂 Projects Included:

1️⃣ Spam SMS Detection 📩
Classifies SMS messages as Spam or Not Spam
Model: Multinomial Naive Bayes
Technique: TF-IDF Vectorization

2️⃣ Credit Card Fraud Detection 💳
Detects fraudulent credit card transactions
Model: Machine Learning classification model (e.g., Random Forest / Logistic Regression)

3️⃣ Customer Churn Prediction 🏦
Predicts whether a bank customer will leave or stay
Helps businesses improve customer retention strategies

🛠️ Tech Stack
Python 🐍
Pandas, NumPy
Scikit-learn
Streamlit (for UI)
Matplotlib (if used for visualization)

🎯 Objective

To build real-world ML solutions for classification problems and deploy them in simple interactive applications.
